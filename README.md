##  Project Description

In this project, I have explored Python as an open-source software and tried to understand its importance in today’s world. The focus is not only on what Python does, but also on why it was created, how its open-source nature helps developers, and how it compares with proprietary tools.

Along with the theoretical analysis, I have also written five shell scripts to demonstrate basic Linux and scripting concepts. These scripts helped me understand how open-source tools work at the command line and how automation can simplify everyday tasks.

##  Scripts Overview

### 1. System Identity Report

This script displays basic system information such as kernel version, current user, uptime, and operating system details. It acts like a simple system introduction screen.

### 2. FOSS Package Inspector

This script checks whether Python is installed on the system and shows its details. It also prints a short description based on the package name.

### 3. Disk and Permission Auditor

This script scans important system directories and shows their permissions and disk usage. It helps in understanding how Linux manages files and access control.

### 4. Log File Analyzer

This script reads a log file and counts how many times a specific keyword appears (like "error"). It also displays the last few matching lines.

### 5. Open Source Manifesto Generator

This is an interactive script that asks the user a few questions and generates a small personalized statement about open-source philosophy.

##  How to Run Scripts

First, give execution permission to all scripts:

```bash
chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh
```

Then run them one by one:

```bash
./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh
```
